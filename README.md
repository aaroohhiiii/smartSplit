# Splitwise

Splitwise is a simple and efficient expense-sharing application that helps groups of people track and split expenses. Whether you're sharing bills with roommates, planning a group trip, or managing shared finances, Splitwise makes it easy to keep everything organized.

## Features

- **Add Expenses**: Record expenses with details like amount, description, and participants.
- **Split Costs**: Automatically split expenses among group members.
- **Track Balances**: View who owes whom and settle up easily.
- **User-Friendly Interface**: Simple and intuitive design for seamless use.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:
 

2. Navigate to the project directory:
   ```bash
   cd splitwise
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

## Usage

1. Open the application in your browser at `http://localhost:3000`.
2. Create a group or join an existing one.
3. Add expenses and let the app handle the calculations.
4. View balances and settle up with group members.

## Technologies Used

- **Frontend**: React, TypeScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Other Tools**: Redux, Axios
